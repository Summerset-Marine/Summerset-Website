SITEMAP 4 — sitemap-images.xml:
Google Image sitemap format. Placeholder structure now — image entries added as photography is loaded. Format per Google spec:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9"
        xmlns:image="http://www.google.com/schemas/sitemap-image/1.1">
  <url>
    <loc>https://summersetmarine.com/markets/lake-geneva/geneva-lake</loc>
    <image:image>
      <image:loc>https://summersetmarine.com/images/[image-filename]</image:loc>
      <image:title>[Alt text string]</image:title>
      <image:caption>[Project description]</image:caption>
      <image:geo_location>Geneva Lake, Lake Geneva, Wisconsin</image:geo_location>
    </image:image>
  </url>
</urlset>
```

SITEMAP INDEX — sitemap-index.xml:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<sitemapindex xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <sitemap>
    <loc>https://summersetmarine.com/sitemap-core.xml</loc>
    <lastmod>[BUILD_DATE_ISO]</lastmod>
  </sitemap>
  <sitemap>
    <loc>https://summersetmarine.com/sitemap-markets.xml</loc>
    <lastmod>[BUILD_DATE_ISO]</lastmod>
  </sitemap>
  <sitemap>

    <lastmod>[BUILD_DATE_ISO]</lastmod>
  </sitemap>
  <sitemap>
    <loc>https://summersetmarine.com/sitemap-images.xml</loc>
    <lastmod>[BUILD_DATE_ISO]</lastmod>
  </sitemap>
  <sitemap>
    <loc>https://summersetmarine.com/sitemap-blog.xml</loc>
    <lastmod>[BUILD_DATE_ISO]</lastmod>
  </sitemap>
</sitemapindex>
```

Every URL in every sitemap must include:
- <loc> — the full absolute URL
- <lastmod> — ISO 8601 date, set to build time for static pages, set to Sanity updatedAt for blog posts
- <changefreq> — as specified above
- <priority> — as specified above

BUILD src/components/seo/PageMeta.jsx:

This component is included on every page. It injects into the document <head>:
- <title> — page-specific, under 60 characters
- <meta name="description"> — page-specific, 120–155 characters
- <link rel="canonical"> — absolute URL for this page
- <meta property="og:title">
- <meta property="og:description">
- <meta property="og:url">
- <meta property="og:type"> — "website" for most pages, "article" for blog posts
- <meta property="og:image"> — page-specific hero image URL
- <meta name="twitter:card"> — "summary_large_image"
- <meta name="twitter:title">
- <meta name="twitter:description">

Props: { title, description, canonical, ogImage, ogType }
