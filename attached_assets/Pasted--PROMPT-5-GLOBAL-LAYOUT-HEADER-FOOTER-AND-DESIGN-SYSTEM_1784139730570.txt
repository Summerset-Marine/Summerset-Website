BUILD src/components/ui/Button.jsx:

Three variants: primary, secondary, ghost.

primary: red (#C8102E) background, white text, rounded (border-radius 6px), padding 12px 24px, font-weight 600, hover darkens background to (#A50D25), transition 150ms. Size variants: default and large (padding 16px 32px, font-size larger).

secondary: steel blue (#1B6A9C) border (2px), steel blue text, transparent background, same radius/padding as primary, hover fills with steel blue background and white text.

ghost: white border (2px), white text, transparent background — for use on dark/image backgrounds. Hover fills white, text turns navy.

Props: { variant: 'primary' | 'secondary' | 'ghost', size: 'default' | 'large', href?: string, onClick?: function, children, className?, type?: string }

If href is provided, render as <a> tag. Otherwise render as <button>. Support type prop for form submit buttons.

BUILD src/components/ui/HeroSection.jsx:

Three variants:

full-bleed: full-viewport-width image background (object-fit cover, object-position center), dark overlay (rgba 0,0,0,0.45), content centered or left-aligned over image. Content: headline (h1, serif font, white, large), subheadline (p, white, semi-transparent), primary CTA button (ghost variant since it's on dark background). Min-height: 85vh on desktop, 70vh on mobile.

split: left half is text content (headline h1, subheadline, primary CTA, secondary CTA), right half is large image (object-fit cover, full height of section). On mobile, image stacks below text. Min-height: 600px desktop.

video-bg: same as full-bleed but accepts a video src. Video autoplay, muted, loop, playsInline. Falls back to poster image if video fails. Used on Home page optionally.

Props: { variant: 'full-bleed' | 'split' | 'video-bg', headline, subheadline, primaryCta: { label, href }, secondaryCta?: { label, href }, imageSrc, imageAlt, videoSrc? }
